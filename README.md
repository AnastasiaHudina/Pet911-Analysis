Содержание репозитория:
- папка "Distribution" - дистрибутив готовой программы, выполняющей анализ данных (запускается в интерактивном режиме);
- папка "Project realization" - представление полного хода выполнения проекта;
- файл "Pet911 Analysis" - полная статья, освещающая ход выполнения проекта;
- файл "Presentation" - презентация проекта.

Данное исследование выполнено в рамках дисциплины «Большие данные» магистерской программы «Математические методы анализа и визуализации данных» СПбПУ Петра Великого (2025г).

Авторы исследования: Худина Анастасия Алексеевна, Буталова Юлия Алексеевна, Цыганков Тимофей Юрьевич, Степанов Андрей Алексеевич.
